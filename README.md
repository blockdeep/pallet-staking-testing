# Instructions

1. Install the dependencies:
   ```bash
   yarn
   ```

2. Install the staking miner:
   ```bash
   cargo install --git https://github.com/paritytech/polkadot-staking-miner polkadot-staking-miner
   ```

3. Run the following command to start zombienet:
   ```bash
   yarn run setup
   ```

4. Run the tests:
   ```bash
   yarn test
   ```
